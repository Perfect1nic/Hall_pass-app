Hall Pass Tracker 🎫
This app is for teachers, it is so that they would be able to make passes for students electronically, they will be able to track how long the student has been gone.

Features ✨
Automatic Starting time
History tab to check which students have gotten a pass
Past history tab to see previous passes
4 automatic choices (Bathroom, Main Office, Nurse, and Another Classroom)
Once pass is submitted it logs the time the student left and in the history tab once the teacher ends it the “timer” stops and shows the time the student returned

Requirements 🪛
In order for this code to work you will need to have Python 3 installed into the device. You also need to download the file in order to run it in Python.
